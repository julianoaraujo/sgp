# levantamentoRequisitos.md

## 1. Informações Gerais
Objetivo, escopo, definições conforme procedimento.

## 2. Fluxo do Processo
Submissão → Avaliação → Viabilidade → Priorização → Consolidação → Validação → Deliberação → Comunicação → Aceite.

## 3. Atores / Perfis
Demandante, SUPRN, Gerente de Projeto, Gerente de Portfólio, Coordenador, Presidência.

## 4. Dados dos Projetos
Título, descrição, justificativa, objetivos, cronograma, recursos, indicadores, documentos.

## 5. Critérios de Avaliação
Etapa A (0-25), Etapa B (0-25), Etapa C (0-60 ponderado).

## 6. Regras de Negócio
Prazos, aprovação, reavaliação, exceções.

## 7. Relatórios e Indicadores
Dashboards, KPIs e relatórios institucionais.

## 8. Controles e Auditoria
Logs, histórico, notificações, rastreabilidade.
